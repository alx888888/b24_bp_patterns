# Дизайн карточки заказа в BBCode

## Цель

Собрать шаблон сообщения для внутреннего мессенджера Bitrix24 в формате карточки заказа:

- крупная шапка с номером выезда, датой и диапазоном времени;
- двухколоночный блок с реквизитами заказа;
- список работ отдельным блоком ниже;
- заметная ссылка на завершение выезда внутри правой колонки.

## Решение

Основой компоновки идет `[table]` с одной строкой и двумя ячейками:

- левая колонка: источник, адрес, домофон / код, подъезд и этаж;
- правая колонка: клиент, комментарий, ссылка завершения.

Блок `Работы:` идет после таблицы обычным списком, чтобы сохранить кликабельные ссылки и не усложнять верстку.

## Причины выбора

- таблица дает визуально устойчивые колонки в мессенджере Bitrix24;
- адрес и служебные данные уходят в левую колонку, клиентский контекст и действие закрытия уходят вправо;
- список работ сохраняет текущую механику и не требует разбиения строк на дополнительные поля.

## Ограничения

- длинный адрес может растянуть левую колонку;
- рендер `[table]` зависит от клиента Bitrix24, поэтому шаблон держит простую структуру без вложенных таблиц;
- блок `Работы` остается текстовым списком, без попытки раскладывать каждую работу по ячейкам.
