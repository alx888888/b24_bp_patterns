# Тестирование методов внутри приложения Rest-Активити

- Раздел: Возможности приложения
- URL: https://it-solution.kdb24.ru/public/restactivity/a299408/
- Последнее изменение: 08.08.2024 14:46:33

В приложении есть встроенная проверка запросов и методов API Битрикс24

1) Проверьте установку последнего обновления (при обновлениях могут добавляться новые методы)

2) Перейдите на вкладку проверка запросов

3) Заполните поле REST-Метод из доступных методов Битрикс24

4) Заполните параметры необходимые для вызова метода

5) Проверьте результат выполнения

Если тестирование метода прошло успешно, то вы можете использовать его в Бизнес-процессах.
