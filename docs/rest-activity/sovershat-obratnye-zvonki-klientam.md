# Совершать обратные звонки клиентам

- Раздел: Примеры использования
- URL: https://it-solution.kdb24.ru/public/restactivity/a328863/
- Последнее изменение: 02.12.2024 10:48:29

Метод voximplant.callback.start запускает обратный звонок клиенту, где после заявки клиента на обратный звонок метод совершает звонок сначала менеджеру, а потом клиенту, соединяя их. Об алгоритме метода можно узнать в [документации по REST](https://dev.1c-bitrix.ru/rest_help/scope_telephony/voximplant/voximplant_callback_start.php).

Метод имеет 4 параметра:

1. FROM_LINE - ID линии, с которой будет выполняться звонок;

2. TO_NUMBER - Номер, на который звонить;

3. TEXT_TO_PRONOUNCE - Текст, который произносится менеджеру перед началом звонка;

4. VOICE - Голос, которым произнести этот текст (необязательный).

С помощью данного метода вы сможете направлять обратные звонки на нужную линию.

! Обратите внимание, направление идет именно на линию. Перенаправлять звонки на ответственного не получится. В методе нет возможности указать ответственного по ID.
