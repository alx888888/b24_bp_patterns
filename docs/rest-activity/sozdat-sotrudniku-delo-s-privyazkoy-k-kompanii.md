# Создать сотруднику дело с привязкой к Компании

- Раздел: Примеры использования
- URL: https://it-solution.kdb24.ru/public/restactivity/a327540/
- Последнее изменение: 02.12.2024 10:29:03

Допустим у вас стоит задача: создавать определенному пользователю Дело с привязкой к определенной Компании.

Для создания Дело с привязкой к определенной Компании вам понадобится метод crm.activity.todo.add.

В параметрах запроса необходимо указать следующее:

```
{
 "ownerId": "2", // id компании
 "ownerTypeId": "4", // Тип элемента, к которому относится дело (4 - компания)
 "description": "Собрание", // Текст дела
 "deadline": "2024-11-21T15:00:00", // Крайний срок дела
 "responsibleId": "1" // Ответственный за дело пользователь
}
```

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/479886/3165/984004/?sig=a4be255ef20e03e5a8bdb262226496f2)

После выполнения запроса в карточке компании, ID которой указано в запросе, появится Дело с ответственным, ID которого также указали в запросе.
