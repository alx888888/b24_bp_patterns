# Статистика и очередь запросов

- Раздел: Обработка запросов
- URL: https://it-solution.kdb24.ru/public/restactivity/a215542/
- Последнее изменение: 10.04.2025 12:13:50

Чтобы посмотреть сколько запросов было выполнено необходимо перейти во вкладку "Статистика запросов" **(1)**.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/277418/3165/560502/?sig=85d45cacda6200f5dd45244c62f58f63)

В данной вкладке Вы можете задать период, за который хотите сформировать отчёт **(2),**(3)****.  

У Вас появится таблица, в которой отображается количество успешных и неуспешных запросов по каждому активити/роботу:  

**(4)** - количество запросов по активити/роботу  **[REST Активити Б24] Получить параметр через REST**  
**(5)**- количество запросов по активити/роботу ****[REST Активити Б24] Получить пользователя через REST****  
**(6)**- количество запросов по активити/роботу ****[REST Активити Б24] Вебхук****  
****(7)** **-**** количество запросов по активити/роботу ****[REST Активити Б24] SnApi****

Также в статистике выводится количество запросов за день и за период ****(8)****.  

Помимо этого, всегда отображается количество запросов за месяц ****(9)****, что позволяет контролировать расход запросов, если у Вас обычная версия.  

  
**ВАЖНО!**

- **Статистика запросов обновляется один раз в сутки в 00:15 МСК. Т.е. статистику за сегодняшний день Вы увидите только на следующий день.**
- **Запросы за месяц ****(9)****отображаются в режиме реального времени.**

Чтобы отслеживать запросы сегодняшнего дня, можно воспользоваться вкладкой "Очередь запросов" ****(10)****.

В данной вкладке запросы отображаются в режиме реального времени. Также здесь Вы можете посмотреть более подробную информацию по запросу.  

**Замечание:**  
**В Статистику запросов и в Очередь запросов попадают только те запросы, которые сделаны через активити/роботов.**  
**Запросы, сделанные во вкладке "Проверка запросов", в статистике не учитываются.**
