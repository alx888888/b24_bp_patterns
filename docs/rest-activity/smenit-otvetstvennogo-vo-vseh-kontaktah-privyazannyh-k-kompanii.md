# Сменить ответственного во всех Контактах, привязанных к Компании

- Раздел: Возможности приложения
- URL: https://it-solution.kdb24.ru/public/restactivity/a89309/
- Последнее изменение: 07.08.2025 12:58:52

Задача: При смене ответственного у "Компании" в Битрикс 24, нужно автоматически сменить ответственного во всех Контактах привязанных к Компании.

Для решения задачи используется приложение "REST Активити Б24"

## Пример бизнес-процесса для решения задачи

Задача будет решаться путем получения ID контактов, где нужно внести изменения и изменения ответственно на того, кто указан в компании в итераторе.

Для решения задачи нам понадобится бизнес-процесс, который будет содержать блоки, представленные ниже.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/585433/3165/1202224/?sig=d53f93570ca7603f0e803a1d2ec2e725)

Для хранения списка контактов для изменения нам понадобится числовая переменная (множественна) для хранения ID контактов.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/585435/3165/1202227/?sig=79f77e3b8af1bcd09abbab503484e3b9)

В первом блоке мы получаем список ID контактов. где нужно поменять ответственного - берем все контакты, которые привязаны к компании.

Для этого используем метод crm.contact.list, откуда с помощью JSONPath берем ID: "$..ID".

В качестве параметров передаем следующее:

```
{
    filter:{
        "COMPANY_ID": {{ID}},
        "!ASSIGNED_BY_ID":{{Ответственный > int}}
    },
    select:['ID']
}
```

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/585434/3165/1202226/?sig=cb286a4e2a4c655a7b41beb92b1d9a7b)

Во втором блоке мы записываем результат кубика "Получить параметры через REST" в ранее созданную переменную. В переменной будет массив ID контактов с привязкой к выбранной компании, где необходимо изменить ответственного.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/585436/3165/1202228/?sig=d4ec7b7e5e71819a2bd2fdd48325827e)

Итерироваться мы будем по переменной, в которой записаны ID контактов.

На каждом шаге итератора для контакту по ID мы будем менять ответственного, который указан в привязанной компании.

Для этого используется метод crm.contact.update и в параметры передается следующее:

```
{
"id": {=A27054_62550_94158_19982:Value}, // ID контакта
"fields": {"ASSIGNED_BY_ID":{{Ответственный > int}}}
}
```

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/585438/3165/1202231/?sig=d870d02cac2e8cd86d1f0e6fb2fc4aba)
