**1. Поиск начальника подразделения**

Выбираем действие в редакторе шаблонов![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/19761/3165/105815/?sig=2c1b80ef17e606c41240178e56d3fe71)

Указываем:

- метод (в примере department.get);

- поле, по которому сортируются результаты и значение этого поля (в примере ищем подразделение с id 8);

- значение какого параметра,  хотим видеть в результате (в примере UF_HEAD - руководитель подразделения).

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/19764/3165/105834/?sig=a27ff0ffc6e53769bcfa51b2731d652c)
