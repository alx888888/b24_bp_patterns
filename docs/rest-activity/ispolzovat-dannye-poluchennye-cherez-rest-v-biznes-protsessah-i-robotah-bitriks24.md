# Использовать данные, полученные через REST, в бизнес-процессах и роботах Битрикс24

- Раздел: Возможности приложения
- URL: https://it-solution.kdb24.ru/public/restactivity/a89304/
- Последнее изменение: 17.01.2025 11:27:08

**Описание**

Создавайте более точные бизнес-процессы, добавляя в них данные, полученные с помощью REST запросов!

**Задача**

Использовать данные, полученные через REST, в бизнес-процессах и роботах Битрикс24.

**Решение**

С помощью приложения REST Активити вы сможете получать параметры  и пользователей. Установите приложение и получите дополнительные блоки бизнес-процессов, которые появятся в разделе Действия Приложений. С ними вы сможете добавлять в бизнес-процесс данные, полученные с помощью REST-запросов.

Ссылка на приложение <https://www.bitrix24.ru/apps/?app=itsolutionru.restactivity>

**Доступно на:**

Коробочном Битрикс24;

Облачных тарифах, поддерживающих работу бизнес-процессов. Подробнее о тарифах можно узнать по [ссылке](https://www.bitrix24.ru/prices/compare_cloud_plans.php).
