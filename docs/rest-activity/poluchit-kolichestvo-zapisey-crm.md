# Получить количество записей CRM

- Раздел: Примеры использования
- URL: https://it-solution.kdb24.ru/public/restactivity/a344698/
- Последнее изменение: 07.02.2025 10:43:41

С помощью метода crm.item.list можно получить список определенных элементов CRM.

С использованием активити приложения можно получать большое количество записей, активировав настройку "Списочный метод". Данная настройка необходима для вывода более чем 50 записей. С использованием этой настройки может понадобится выводить количество элементов в списке. Количество записей может возвращаться в значении total в ответе на запрос REST Активити.

В случае, если количество элементов не выводится, то вывести количество элементов в списке можно с помощью следующего JSON-Path: $.items[length()]
