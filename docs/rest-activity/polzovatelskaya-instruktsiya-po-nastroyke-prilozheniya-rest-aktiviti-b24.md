## Описание приложения

По умолчанию бизнес-процессы в Битрикс24 используют формат JSON при задании параметров для REST-запросов. Синтаксис JSON довольно сложный, и поскольку поля в параметрах бизнес-процессов не имеют проверки на корректность записи, искать опечатки и ошибки достаточно не удобно. Приложение REST Активити Б24 от IT-Solution позволяет использовать при заполнении параметров бизнес-процессов не только JSON, но и форматы с более простым синтаксисом: YAML, TOML, XML.

Для работы приложения потребуются как минимум роботы, соответственно, тариф портала Битрикс24, на который устанавливается приложение, может быть любым. Приложение распространяется по [Подписке на Маркет](https://it-solution.ru/bitriks24_market/) и имеет ограничение в 3000 запросов в месяц. Запросы обнуляются 1-го числа каждого месяца. Чтобы уточнить стоимость снятия такого ограничения, обращайтесь [в чат поддержки](https://b24.it-solution.ru/online/?IM_DIALOG=7321).

## Установка приложения в Битрикс24

Для того чтобы начать использовать приложение «REST Активити Б24» на вашем портале Битрикс24, необходимо скачать его по [**ссылке**](https://www.bitrix24.ru/apps/app/itsolutionru.restactivity/) или выполнить следующие действия:

## 1. Поиск приложения

Перейдите в раздел Маркет вашего портала Битрикс24, в строке поиска введите название «REST Активити Б24» и выберите его из предложенного списка.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/575931/3165/1189559/?sig=2a4d56934c069c6bb97aa94ff33a324b)

## 2. Открытие страницы приложения

После выбора приложения вы попадете на страницу с его описанием, где можно ознакомиться c функционалом и возможностями.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/575932/3165/1189560/?sig=c3ae39a8952eafeae0fb646e2277f62d)

## 3. Установка приложения

Нажмите кнопку «Установить», поставьте галочки в чекбоксах условий соглашения, а затем снова нажмите на кнопку «Установить».

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/575933/3165/1189562/?sig=c0b7bdffcf922a419e70a34fa1c7f8c1)

## 4. Доступ к приложению

После завершения установки приложение будет доступно в главном меню вашего портала Битрикс24.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/591011/3165/1211006/?sig=dd2da75bd706f707d558ff7900f40b08)

## Функционал приложения

## Бизнес-процессы

1. В конструкторе бизнес-процессов появится блок «Действия приложений» (в него можно перейти с помощью стрелочку в самом конструкторе, а также через правую панель), откуда можно выбрать любое из действий приложения: SnapApi, Вебхук, Получить параметр или Получить пользователя.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/474375/3165/975228/?sig=9e8b82a3be1d762401356c5ac3c84dd2)

2. В настройке параметра действия можно ввести REST-запрос в любом из доступных форматов: json, yaml, xml или toml. Выберите метод в поле «REST-метод» и введите запрос.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/474376/3165/975229/?sig=f3b58b5c374d535e4f73d6080c42329d)

***Обратите внимание****, если в настройке параметров действий в поле «Списочный метод» из раскрывающегося списка выбрать «Нет», то будет включено ограничение — за один раз можно получить не более 50 записей. Если выбрать «Да», то за один раз можно получить неограниченное количество записей.*

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/474368/3165/975220/?sig=b52d4f4ee56603cc069ca3441ff9d102)

## Роботы

1. Перейдите в роботы → «Другие роботы», выберите необходимый и нажмите «Добавить».

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/561100/3165/1167886/?sig=1e51d90a95354e7b63bf086103e8a283)

2. Заполните запрос аналогично действию бизнес-процесса.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/474372/3165/975226/?sig=68ca29386d1a6fb578adef68c4f1afc1)

Результат можно использовать в следующем роботе, который находится на той же стадии. Для этого перейдите в «Роботы», нажмите на кнопку «+» на нужной стадии. Затем выберите необходимого робота из списка и нажмите «Добавить». В появившемся окне рядом с полем «Текст сообщения» нажмите на троеточие, выберите из выпадающего списка робота и нажмите «Сохранить».

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/474371/3165/975224/?sig=38551683d756a319021f346f1da2b01b)
