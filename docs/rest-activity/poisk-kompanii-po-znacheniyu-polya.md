# Поиск компании по значению поля

- Раздел: Возможности приложения
- URL: https://it-solution.kdb24.ru/public/restactivity/a89312/
- Последнее изменение: 28.12.2024 17:42:36

**В компании и сделки есть поле ИНН, необходимо найти компанию с ИНН указанным в сделке и привязать компанию к сделке.**

1) Составляем REST запрос  
1.1) Указываем необходимый REST-метод;  
1.2) Указываем параметры REST-запроса;  
Указываем информацию которую хотим получить.

В нашем случае:

1.1) метод  "crm.company.list" (фильтрация компаний по значению);  
1.2) указываем по какому полю фильтруем и какое значение должно быть у поля;  
1.3) указываем, что в ответе хотим получить id компании;

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/25472/3165/114716/?sig=b754bea20c6d69b3cd6f56eb98dd4aae)

2) Создаем действие "изменение документа", указываем что хотим изменить поле "Компания" на результат сохранный в разделе "Дополнительные результаты".
