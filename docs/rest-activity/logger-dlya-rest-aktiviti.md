# Логгер для REST Активити

- Раздел: С чего начать?
- URL: https://it-solution.kdb24.ru/public/restactivity/a132133/
- Последнее изменение: 07.08.2025 12:57:14

Собирайте подробнейшие логи о работе приложения REST-Активити в чат Телеграм! Создайте любой чат в Телеграм, пригласите туда бота @restactivity_bot и ваш удобный логгер готов к работе. Больше видео о возможностях REST-Активити можно посмотреть [в этом плейлисте.](https://youtube.com/playlist?list=PLOEsyFAuoTfVt0iU6f8KLZ-csD25OhnPe&si=dk1f_Zmcj6cvZ6s1)

В приложении перейдите на вкладку "Настройки" и привяжите бота @restactivity_bot по ссылке к созданному ранее чату Телеграм.

![](https://it-solution.kdb24.ru/api/v1/froala/file_proxy/362066/3165/717573/?sig=a8dd226f01db88600db42a902cd83c93)

В чат будут сохраняться все логи о вызовах REST Битрикс24 с помощью приложения. Проверка запросов в лог собираться не будет.
